
**项目描述：**

> 很多人记不住文档的路径，但却能记住文件的标签；
> 这个工具就是帮助人们方便的根据标签来查找文档。

**当前状态：**

> 当前项目版本是bata1.0；
> 目前只发布给开发人员；
> 项目是用C#开发的；使用了.NET 4.5；界面用WPF实现的；IDE是VS2013 UPDATE4
> 数据库是sqlite；

**使用帮助**

> 开发人员可以下载代码，编译运行就能看到界面；
> 可以把“右键菜单注册表.reg”导入到注册表中，所有文件和文件夹就多了一项右键菜单：DocTag
> 用户可以通过此菜单，来为文件和文件夹添加标记
> 数据库在debug目录下

**产品规划**

>  - 根据标签查找文档
>  - 添加标签，搜索标签时增加智能提示功能
>  - 购买云服务，让用户的标签可以保存在云服务器上
>  - 增加注册登录功能，便于用户把本地数据保存到服务器上
>  - 增加版本自动更新功能，用户使用本软件可以自动更新到最新版本
