
**项目描述：**

> 很多人记不住文档的路径，但却能记住文件的标签；
> 这个工具就是帮助人们方便的根据标签来查找文档。

**当前状态：**

> 当前项目版本是bata1.0；
> 目前只发布给开发人员；
> 项目是用C#开发的；使用了.NET 4.5；界面用WinForm实现的（WPF实在是太复杂了）；IDE是VS2013 UPDATE4；
> 数据库是sqlite；

**使用帮助**

> 开发人员可以下载代码，编译运行就能看到界面；
> 可以把“右键菜单注册表.reg”导入到注册表中（注意要修改一下里面的路径），所有文件和文件夹就多了一项右键菜单：DocTag
> 用户可以通过此菜单，来为文件和文件夹添加标记
> 数据库在debug目录下
